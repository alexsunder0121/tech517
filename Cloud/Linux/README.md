## Why learn linux 

* Fast-growing 
* Inexpensive (Red Hat enterprise version of linux does cost)
* Flexible, scales (down) well 
* Often used in DevOps
* Stability - does it crash?


## What is Linux

* A spinoff of Unix OS
* Unix OS used to be used on large mainframes 
* Linux is now able to run on small/large computers now
* Kernel - it is the core OS, plus libriaries/utilities that rely on the kernel

## Processes
* what is a process? 
  * A program that is loaded into RAM and could be using CPU
  * Two types of processes in linux 
    * 1. User Processes - 
      * Usually run by a user and linked to the shell session in which they were run
    * 2. System Processes - 
        * Most of them running
        * Dont usually provide an application or interface for the end user
